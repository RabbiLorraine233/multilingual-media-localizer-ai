## Core Features

- **Dual-Mode Generation**: Instantly transforms raw text into a faithful Direct Translation and a high-converting E-commerce Copy.
- **Cross-Cultural Adaptation**: Bridges communication gaps with tailored support for multiple languages, including English, Thai, and Indonesian.
- **Responsive Architecture**: Fluidly adapts to desktop, tablet, and mobile screens to ensure a seamless experience on any device.
- **Privacy First**: Requires a local API key input for generation, ensuring your credentials are never stored or tracked.

## Technical Implementation

- **Frontend Structure**: Semantic HTML5 and native CSS3, utilizing flexbox layouts and micro-interactions (such as hover states and smooth transitions).
- **Core Logic**: Vanilla JavaScript (ES6+) handling DOM manipulation, state management, and asynchronous operations.
- **AI Integration**: Built with the Fetch API to communicate securely with the DeepSeek Large Language Model, executing complex prompt engineering.

## Usage Guide

1. **Secure Access**: Enter your personal API Key in the designated secure input field.
2. **Input Content**: Paste your original text into the left source content area.
3. **Select Target**: Choose your desired target language from the dropdown menu.
4. **Generate & Copy**: Click the generate button. Once the AI finishes crafting the content, simply copy the results from the customized cards.
